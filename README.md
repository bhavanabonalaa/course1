# 🎓 Course Selection and Teacher Assignment System

A full-stack web application that allows university students to select *4 theory* and *2 lab* courses each semester, assigning a teacher to each course. The platform provides secure login/register functionality, detailed teacher profiles, course filtering, feedback submission, and toast notifications for better user experience.

## 📦 Features

✅ Student Registration & Login  
✅ Protected Routes (Courses, Feedback, Teacher Profiles)  
✅ Select 4 Theory & 2 Lab Courses  
✅ Assign Teachers from Dropdown List  
✅ View Detailed Teacher Profiles with Ratings, Research & Patents  
✅ Submit Feedback on Courses & Teachers  
✅ Toast Notifications for Success/Error  
✅ Fully Responsive Design  
✅ Backend with Express.js, MongoDB  
✅ Frontend with React, Axios, React-Router
